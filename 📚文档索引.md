## 📝 工作流程

### 创建新指南的完整流程

```bash
# 1. 查看分类
node show-all-categories.js

# 2. 复制模板
cp add-guide-template.js add-新指南.js

# 3. 编辑内容
code add-新指南.js

# 4. 运行创建
node add-新指南.js

# 5. （可选）配置置顶
# 编辑 frontend/lib/directus.ts

# 6. （可选）添加图片
# 在 Directus 后台上传

# 7. 验证
open http://localhost:3000/strategies/你的-slug
```
