- Email: `the_uk1@outlook.com`
- Password: `Mygcdjmyxzg2026!`

### 2. 进入权限设置
左上角 **⚙️ 设置** → **Access Control**

### 3. 配置 Public 角色
1. 点击 **"Public"** 角色
2. 找到 **"Directus Files"**
3. 启用 **✅ Read** 权限
4. 设置：
   - Access: **All Items**
   - Fields: **All Fields**
5. **保存**

### 4. 重启 Directus
```bash
docker-compose restart directus
```

### 5. 测试
访问（无需登录）：
```
http://localhost:8055/assets/{任意文件ID}
```
