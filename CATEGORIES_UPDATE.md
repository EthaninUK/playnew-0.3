# 分类系统更新

## 新的分类体系

### A1. 撸空投&积分 (airdrop-points) 🎁
**说明**: 通过完成任务、交互协议等方式获得空投和积分奖励

**包含的玩法**:
- Arbitrum 生态空投挖掘
- zkSync Era 测试网交互
- Galxe 任务空投挖掘

**特点**: 
- 成本低或零成本
- 回报不确定但潜力大
- 需要持续投入时间

---

### A2. 稳健赚利息 (stable-yield) 🏦
**说明**: 低风险的稳定收益策略，如质押、存款等

**包含的玩法**:
- Lido 质押 ETH 赚取收益
- Curve 稳定币流动性挖矿

**特点**:
- 低风险
- 收益稳定可预期
- 适合长期持有
- 适合新手

---

### A3. 做市赚手续费 (market-making) 💱
**说明**: 通过提供流动性赚取交易手续费

**包含的玩法**:
- Uniswap V3 集中流动性挖矿

**特点**:
- 需要主动管理
- 存在无常损失风险
- 收益与交易量相关
- 需要一定经验

---

### A4. 对冲/套利 (arbitrage-hedging) ⚖️
**说明**: 通过价差套利或对冲策略获利

**包含的玩法**:
- Aave V3 循环借贷套利

**特点**:
- 需要深入理解 DeFi 机制
- 风险较高
- 需要密切监控
- 高级用户专属

---

### A5. 进阶衍生品 (advanced-derivatives) 📈
**说明**: 期权、永续合约等高级衍生品策略

**包含的玩法**:
- GMX 永续合约做市
- Pendle 固定收益交易

**特点**:
- 复杂度高
- 潜在收益高
- 需要专业知识
- 适合高级用户

---

## API 访问

```bash
# 获取所有分类
curl http://localhost:8055/items/categories?sort=order_index

# 按分类筛选玩法
curl 'http://localhost:8055/items/strategies?filter[category][_eq]=airdrop-points'

# 获取某个分类下的玩法数量
curl 'http://localhost:8055/items/strategies?aggregate[count]=id&filter[category][_eq]=stable-yield'
```
