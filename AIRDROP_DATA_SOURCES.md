## 💡 数据去重和质量控制

### 去重策略

1. **项目名称 + 链 ID** 作为唯一标识
2. **URL 作为次要标识**
3. **时间窗口去重**（同一项目 7 天内只保留最新）

### 质量评分

```javascript
quality_score = (
  source_reliability * 0.3 +     // 数据源可信度
  project_funding * 0.2 +         // 项目融资情况
  social_engagement * 0.2 +       // 社交媒体热度
  task_complexity * 0.15 +        // 任务复杂度
  reward_value * 0.15             // 奖励价值
)
```

### 风险标记

- 🔴 高风险：无融资、匿名团队、要求私钥
- 🟡 中风险：新项目、小额融资
- 🟢 低风险：知名 VC、公开团队、大额融资
